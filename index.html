Kenjsplayer.audio

Audio Player 100% HTML5 + CSS + JS and 0% FLASH or SWF
download .ZIPdownload .TGZ

Welcome to KenjsPLayer Pages.

This is a automatic audio player generator that don't use .swf files, or any flash files. .The API goal in jQuery is to eliminate the use of flash and nothing but javascript to manipulate the player.For the perfect functioning were used the following frameworks and tools :

  • Globo Bootstrap
  • Font Awesome
  • jQuery
